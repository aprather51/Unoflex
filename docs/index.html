    Grid

    Using Flexbox grid system to render fluid reponsive screen for easy and flexible use.

    Set Rules on Grid System

    • .container should be placed at the first part of body for proper screen alignment and padding.
    • .row should be placed inside .container to display multiple columns in group.
    • Create many .row as much necessary to separate columns in group.
    • .col should be placed inside .row and it can be as much as necessary (Recommend Eight to Ten column which will suffice).

    Fluid Grid

    Mixin & Variable

    The default button is already set up with sass @mixin Use variable (i.e, $bgColor, $color, $radius, and $border ) to make a custom change.

    Mixin Description
    $bgColor Set any background color on button Note: Outline button Background will always be transparent except for on hover.
    $color Set any text color on button
    $hoverColor Set hover text color on button (For Outline Button Only)
    $borderRadius Transform depth corner of button from Hard corner to pill-like.
    $borderRadius Set border thickness, color and etc. surround button.

    Example on mixin and variable used in SCSS that gives result to default buttons above.

    
                    $bgColor: #d3d3d3; 
                    $color: #454545;
                    $radius: 0rem;
                    $border: 2px solid $bgColor;

                    .btn{
                        @include btnSolid($bgColor, $color, $radius, $border);
                    }

    Although there are two sets of default buttons that can be used:

    1. @include btnSolid - Solid Button with background.
    2. @include btnOutline - Border button with invisible background.

    Custom Button

    The button can be customized. To do this, create separate scss file i.e. _custom.scss or any scss file name. Include @import "./components/button" at the top of file, customize the variable name with any value and button class name of your preference. see example below.

    Button: Solid

    HTML

    
                    <button class="mySolidButton">Button</button>

    SCSS Customization example for Solid Button inside _custom.scss

    
                    @import "./components/button"

                    $bgColor: #f00;
                    $color: #fff;
                    $radius: 0rem;
                    $border: 2px solid $bgColor;

                    /* Custom Class Name */
                    .mySolidButton{
                        @include btnSolid($bgColor, $color, $radius, $border);
                    }

    Result for solid Button


    Button: Outline

    HTML

    
                    <button class="myOutlineButton">Button</button>

    SCSS Customization example for Outline Button inside _custom.scss

    
                    @import "./components/button"

                    $bgColor: #00F;
                    $color: $bgColor;
                    $hoverColor: #fff;
                    $radius: 0rem;
                    $border: 2px solid $bgColor;
                    
                    /*Custom Class Name*/
                    .myOutlineButton{
                        @include btnOutline($bgColor, $color, $hoverColor, $radius, $border);
                    }

    Result for outlineButton
