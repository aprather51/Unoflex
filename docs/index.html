    Grid

    Flexbox grid system to render fluid reponsive screen for easy and flexible use.

    Rules of grid system

    • The .container should be placed at the first part of body for proper screen alignment and padding.
    • The .row should be placed inside .container to display multiple columns in group
    • Create another .row If necessary to separate columns in group
    • With flexbox .col can be added as many inside .row (Eight to Ten column will suffice).

    Fluid Grid

    An column with default 100% width size and any number of column that is added within inside of .row will automatically divide by 100% to get an equal width based on device screen width and column size.

    Spacing

    There are no spacing anywhere in grid system. Although, it can be customized in css style at your preference. The default grid without spacing allow you to have more options in styling the grid.

    Spacing between Rows

    The example below shown custom class name .row-space contain margin size at .5rem. Renders spacing between row (horizon).

    100%
    50%
    50%
    33%
    33%
    33%

    CSS

    
                   .row-space{ margin: .5rem}

    HTML

    
                <div class="row col-space">
                    <div class="col col-space ">25%</div>
                </div>
    Spacing Between Column

    Another example below shown class name .col contain margin size at .5rem. Renders spacing around column (box).

    100%
    50%
    50%
    33%
    33%
    33%

    CSS

    
                .col-space{ margin: .5rem}

    HTML

    
                <div class="row col-space">
                    <div class="col col-space ">25%</div>
                </div>
